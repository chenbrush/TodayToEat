# TodayToEat \- 今天吃什么

**一款简洁、高颜值、本地离线的「随机干饭」工具**

解决选择困难症，不知道今天吃什么？交给程序帮你随机决定！

---

## 项目介绍

**TodayToEat** 是一款纯本地离线的 Android 小工具，专为解决大学生「选择困难、每天不知道吃什么」的问题开发。

你可以自由添加、编辑、管理自己常吃的商铺，软件会智能随机生成：

- 今日中饭

- 今日晚饭

- 下一餐随机推荐

全程**无联网、无广告、无后台请求**，所有数据保存在本地，安全纯净。

---

## 核心功能

- **自定义商铺库**：自由添加、删除、修改常吃店铺

- **智能随机算法**：自动规避昨天吃过的店铺，避免重复干饭

- **双模式随机**

    - 下一餐（适合午后、宵夜）

    - 全天三餐规划（中饭 \+ 晚饭）

- **时间智能判断**：根据当前时间动态切换文案与推荐策略

- **本地文件持久化**：商铺列表、每日记录全部本地保存
